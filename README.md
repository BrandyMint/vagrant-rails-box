== Настройка

1. Установить vagrant 1.2.2
2. `vagrant plugin install vagrant-berkshelf`

Установка виртуалки:

Под linux: `vagrant plugin install vagrant-lxc`
Под Mac: `установить virtialbox`


```
> bundle; berks
```

== Запуск

Под linux: `vagrant up --provider=lxc`
Под Mac: `vagrant up`
